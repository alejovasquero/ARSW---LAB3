#### Instalación

Vamos a compilar el código con maven.
   ```console
mvn compile
   ```


Para ejecutar el programa.

  ```console
mvn exec:java -Dexec.mainClass="edu.eci.cinema.SpringCinema"
   ```

Para ejecutar las pruebas

  ```console
mvn test
   ```
